A) HTML Structure and css structure - top

This theme is responsive design. i use Bootstrap 3.0 in all of my code in this project , you can change the colors and images as you wand as following :

for change the color just go to css file and find what you want to change and write your new color in hexadicimal , ex: from old color white

#primaryContent a {
	      	color: #fff	;
	       }

to new color black 

#primaryContent a {
		      color: #000	;
	         }

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

	#wrap #primaryContent a {
		color: #someColor;
	}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.
